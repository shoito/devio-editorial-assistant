## カテゴリと特集カテゴリのリアルタイムフィルタリング
記事コンテンツの増加に伴いカテゴリと特集カテゴリの数も増加してきました。  
記事編集画面にて、それらをフィルタリングする機能を追加しカテゴリの選択をしやすくします。
